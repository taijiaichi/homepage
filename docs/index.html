◆武術太極拳とは？◆

数千年という長い歴史をもつ｢中国武術｣。広大な中国各地において様々な進化を遂げ、数多くの武器・武術の流派が生まれました。 打つ・蹴る・投げるなど…、もともと戦う手段であった中国武術も、現代においては健康法や競技スポーツといったように、その目的も多様化しながら発展してきました。 古くから｢中国武術＝KUNG-FU・カンフー｣ という言葉で一括りされていましたが、現在中国を始め国際的には、｢武術(WUSHU・ウーシュウ)｣と呼ばれています。そして日本国内においては 『武術太極拳』という名称で普及が進んでいます。 健康法としてすっかり定着している｢太極拳｣や、競技スポーツとしての｢カンフー(長拳)｣など…、ジュニアから高齢の方々まで、老若男女問わず幅広い世代の皆さんが活動しています。

◆代表的な３種類の拳術◆

歴史ある 『武術太極拳』 には数多くの拳術・武器があり、中でも 『太極拳』 『長拳』 『南拳』 は代表的な拳術（現在国際競技種目となっている３種類の拳術）といえます。それぞれの拳術には武器種目もあり、当協会ではこれらの拳術や武器を中心に練習しております。

無理のないゆったりとした動きの中にも、攻防動作を含む 『太極拳』。年代や体力に応じて取り組めるメリットがあり、心身のリラックス効果、足腰の強化によりバランス感覚も養われます。主に健康法として広く普及していますが、競技スポーツやパフォーマンスといった側面もあります。


『長拳』 は、筋力・柔軟性・敏捷性などが必要とされる全身運動で、子どもにとっては偏ることなく基礎体力を高めることができるスポーツといえます。大人にとっても、無理のないペースで取り組むことで武術太極拳全般の基本技術や体力づくりに適しています。


いろいろな拳術の中でも、最も力強く迫力ある 『南拳』 は、足腰や体幹が鍛えられます。流派が多く、動物の動きを取り入れた動作や多様な手技が特徴的で、時に気合いを発したりしたます（発声）。
